**1. Power BI Training**

**1. 1-day workshop at (West Wharf GSK-Pakistan)
2. 4-day basic to intermediate training at (Korangi GSK-Pakistan)**

Conducted structured hands-on training sessions, covering end-to-end Power BI concepts. Trained participants on:

* Data loading and transformation using Power Query
* Data cleaning and shaping techniques
* Data modeling basics (relationships, data types)
* Creating dashboards and reports from scratch
* Applying data visualization best practices (chart selection, layout, storytelling)
* Building interactive reports using slicers, filters, and drill-through
* Writing basic DAX measures for calculations and KPIs
* Publishing and sharing reports via Power BI Service

Sessions included hands-on exercises and real-world business scenarios to help users generate actionable insights and improve decision-making.
